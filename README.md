# Internship Task 2 – Strict Tech Stack Implementation

## Objective
The objective of this task is to recreate the Internship Task 1 document
using a mandatory templating engine and a build process, while ensuring
the final output matches the original document exactly in structure and styling.

---

## Technology Stack Used
- **Templating Engine:** Nunjucks
- **Build Tool:** Node.js (custom build script)
- **Architecture:** Static site generation (Jamstack inspired)
